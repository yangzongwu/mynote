### ordered list
```html
<h1>Lists</h1>
<ol>
    <li>First Item</li>
    <li>Second Item</li>
    <li>Third Item</li>
</ol>
```
### unordered list
```html
<ul>
    <li>First Item</li>
    <li>Second Item</li>
    <li>Third Item</li>
</ul>
```
### nested
```html
<ol>
    <li>Item One</li>
    <li>Item Two</li>
    <ul>
        <li>I am nested in side</li>
        <li>Also nested</li>
    </ul>
</ol>
```
