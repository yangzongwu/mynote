# 条件和条件语句
```py
name = input('What is your name? ')
if name.endswith('Gumby'):
 print('Hello, Mr. Gumby') 
```

```py
name = input('What is your name?')
if name.endswith('Gumby'):
 print('Hello, Mr. Gumby')
else:
 print('Hello, stranger') 
```

```py
num = int(input('Enter a number: '))
if num > 0:
 print('The number is positive')
elif num < 0:
 print('The number is negative')
else:
 print('The number is zero') 
```
