# 循环
### while
```py
x = 1
while x <= 100:
 print(x)
 x += 1
```

### for
```py
words = ['this', 'is', 'an', 'ex', 'parrot']
for word in words:
 print(word) 
```

```py
for number in range(1,101):
 print(number) 
```

```py
d = {'x': 1, 'y': 2, 'z': 3}
for key in d:
 print(key, 'corresponds to', d[key]) 
```
