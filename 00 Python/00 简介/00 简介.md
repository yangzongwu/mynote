# 注释
### 单行注释
* 注释用(#)标识。井号后面的内容都会被Python解释器忽略
* 快捷键：Ctrl+/ 
### 多行注释
* 6个双引号
```python
'''commit'''
```
* 6个单引号  
```python
"""commit"""
```
